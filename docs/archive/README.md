# Archive

This directory stores old proposals, negative results, and historical benchmark notes.

These files are useful for memory, but they are not the active paper plan:

- `benchmark_multimodal.md`: first five-mode benchmark plan for acceleration regularization.
- `benchmark_fan_modes.md`: asymmetric fan benchmark proposal for directional regularization.
- `benchmark_ring_modes.md`: ring/clumped results for directional regularization.
- `burgers_experiment.md`: early Burgers surface-generation proposal; this is obsolete and uses
  old "upwind" language.
- `pressure_training_status.md`: pressure-aware training results and failure modes.
- `pressure_law_and_phase1_history.md`: pressure-law derivation and Phase-1 historical plan.

Current active docs live one level up. In particular, use `../plan.md`, `../literature.md`, and
`../todo.md` for the current research direction.
